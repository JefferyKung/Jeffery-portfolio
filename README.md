## how to download
1. git clone  
2. npm install --legacy-peer-deps  
3. go to sign up for emailJS and create a .env file with  
_VITE_APP_EMAILJS_SERVICE_ID  
VITE_APP_EMAILJS_TEMPLATE_ID  
VITE_APP_EMAILJS_PUBLIC_KEY_  
https://www.emailjs.com/  
4. npm run dev
